Contact submissions

Lists messages stored by your contact-backend API on DigitalOcean (or elsewhere). Third-party form hosts (Formspree, Web3Forms) use their own dashboards instead — see hints below.

Received Name Email Interest Message

No self-hosted API? Use Formspree (set CONTACT_FORMSPREE_ID in index.html) and read submissions in their inbox. Web3Forms emails you directly. PapaWeb home form sends to whichever provider you configure first in that priority chain.

Security: if you use this page with a live API, keep /admin/ off the public web or protect it (e.g. HTTP basic auth on your static host). The token lives only in this browser’s localStorage.
